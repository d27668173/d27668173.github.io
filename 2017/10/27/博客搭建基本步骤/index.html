博客搭建基本步骤

1.配置环境(此版本只针对windows)

安装Node（必须）

作用：用来生成静态页面的

到Node.js官网下载相应平台的最新版本，一路安装即可。

==由于墙的原因,安装完node.js之后最好设置一下淘宝镜像==


npm install -g cnpm –registry=https://registry.npm.taobao.org

# 安装Git（必须）

作用：把本地的hexo内容提交到github上去.

下载小乌龟,一路安装下去

申请GitHub账号以及设置博客仓库（必须）

作用：是用来做博客的远程创库、域名、服务器之类,博客承载平台。

新建博客仓库

new repository=>填写 Repository name
要求是Repository name 保持仓库名为【your_user_name.github.io】的固定写法

2.配置git

1.设置user.name和user.email配置信息：

git config –global user.name “d27668173”

git config –global user.email “1983463947@qq.com”

2.生成ssh密钥文件配置到

ssh-keygen -t rsa -C “1983463947@qq.com

从c:/user/administrator/.ssh文件夹中打开id_rsa.pub密钥，将内容全部复制

点击图像图标=>settings=>SSH keys=>new SSH key=>粘贴在key里面=>填写标题(自定义)

=>add SSH key 完工

3.hexo安装以及初始化

安装完node.js后建立一个项目文件,在文件浏览栏中输入cmd

在弹出的输入框用npm命令安装Hexo，输入：npm install -g hexo-cli

安装完成后，初始化我们的博客，输入：
hexo init blog
==#blog 是项目文件名==

hexo new this_site ==新建博客页==

hexo generate（hexo g也可以） ==生成静态页面==

hexo server(hexo s) ==本地启动==

======一般到这里输入localhost====:4000就能在本地预览自己的博客了==

4.关联github

我们在blog文件下会看到_config.yml文件
打开_config.yml文件,翻到最下面，改成这样子的
deploy:
     type: git    
     repo: https://github.com/d27668173/d27668173.github.io.git
     branch: master
保存之后在cmd命令框中执行cnpm install hexo-deployer-git --save
完成之后    执行配置命令：
hexo deploy (hexo d也成) 会自动更新github上的博客内容

5.部署步骤

每次部署的步骤，可按以下三步来进行。

hexo clean

hexo generate

hexo deploy

6.hexo 常用命令：

hexo new”postName” #新建文章

hexo new page”pageName” #新建页面

hexo generate #生成静态页面至public目录

hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）

hexo deploy #将.deploy目录部署到GitHub

hexo help # 查看帮助

hexo version #查看Hexo的版本

so easy!
